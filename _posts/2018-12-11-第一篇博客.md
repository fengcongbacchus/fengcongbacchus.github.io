# 一. Dart介绍：

>Dart是由谷歌开发的计算机变成语言，它可以被用于web、服务器、移动应用和物联网等领域的开发
Dart诞生于2011年，号称要取代JavaScript。但是过去的几年中一直不温不火。直到Flutter的出现，现在被人们重新重视起来。
要学Flutter的话，我们必须首先得学会Dart。


# 二. Dart环境搭建


>要在我们本地开发Dart程序的话，首先需要安装Dart SDK  
官方文档：[获取Dart SDK](https://dart.dev/get-dart)

### windows(推荐)：
[获取Dart SDK](http://www.gekorm.com/dart-windows/)

### MAC:
```shell
brew tap dart-lang/dart
brew install dart
```

# 三. Dart开发工具
>Dart开发工具有很多：Intelij IDEA 、WebStorm 、Atom 、Vscode等
以后我们主要用Vscode
>1. 下载Vscode
2. 找到Vscode插件安装Dart
3. 找到Vscode插件安装Code runner(Code runner可以运行我们的文件)
